# auth-service

Microservicio de autenticacion.

## Rutas

- `POST /auth/login`
- `POST /auth/register`
- `POST /auth/google`
- `GET /auth/me`
- `POST /auth/logout`
- `POST /api/auth/login`
- `GET /api/auth/me`

## Variables de entorno

- `PORT`
- `SPRING_DATASOURCE_URL`
- `SPRING_DATASOURCE_USERNAME`
- `SPRING_DATASOURCE_PASSWORD`
- `GOOGLE_CLIENT_ID`
