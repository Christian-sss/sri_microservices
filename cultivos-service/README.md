# cultivos-service

Microservicio de cultivos.

## Rutas

- `GET /api/cultivos`
- `GET /api/cultivos/activos`
- `GET /api/cultivos/inactivos`
- `GET /api/cultivos/{id}`
- `POST /api/cultivos`
- `PUT /api/cultivos/{id}`
- `PUT /api/cultivos/{id}/toggle-estado`

## Variables de entorno

- `PORT`
- `SPRING_DATASOURCE_URL`
- `SPRING_DATASOURCE_USERNAME`
- `SPRING_DATASOURCE_PASSWORD`
