# reportes-service

Microservicio de reportes y estadisticas.

## Rutas

- `GET /api/reportes/descargar-pdf`
- `GET /api/reportes/consumo-agua`
- `GET /api/estadisticas/resumen`
- `GET /api/estadisticas/telemetria`
- `GET /api/estadisticas/distribucion-modos`
- `GET /api/estadisticas/consumo-detalle`

## Variables de entorno

- `PORT`
- `SPRING_DATASOURCE_URL`
- `SPRING_DATASOURCE_USERNAME`
- `SPRING_DATASOURCE_PASSWORD`
